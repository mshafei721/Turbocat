# Theme Factory Skill

Curated collection of professional font and color themes for styling artifacts.

## Purpose

Apply consistent, professional styling to:
- Presentation slide decks
- Documents
- Reports
- HTML landing pages
- Any visual artifact

Each theme includes:
- Cohesive color palette with hex codes
- Complementary font pairings
- Distinct visual identity

## Usage Instructions

1. **Show theme showcase** to let users see available themes
2. **Ask for choice** of which theme to apply
3. **Wait for selection** with explicit confirmation
4. **Apply the theme** colors and fonts consistently

## Available Themes (10)

1. **Ocean Depths** - Professional maritime theme
2. **Sunset Boulevard** - Warm sunset colors
3. **Forest Canopy** - Natural earth tones
4. **Modern Minimalist** - Clean grayscale
5. **Golden Hour** - Rich autumnal palette
6. **Arctic Frost** - Cool winter-inspired
7. **Desert Rose** - Soft dusty tones
8. **Tech Innovation** - Bold tech aesthetic
9. **Botanical Garden** - Fresh garden colors
10. **Midnight Galaxy** - Dramatic cosmic tones

## Theme Specifications

Each theme includes:
- Primary, secondary, accent colors (hex codes)
- Header font family
- Body font family
- Background colors
- Text colors with proper contrast

## Application Process

1. Read theme file from `themes/` directory
2. Apply colors consistently throughout
3. Apply font families to headers and body
4. Ensure proper contrast and readability
5. Maintain visual identity across all elements

## Custom Theme Creation

If no existing theme fits:
1. Based on user description, generate new theme
2. Choose appropriate colors and fonts
3. Name the theme descriptively
4. Show for review before applying
