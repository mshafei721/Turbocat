## Quick Start

1. Open Turbocat
2. Select "Mobile" from the platform dropdown
3. Describe the app you want to build
4. Scan the QR code with Expo Go on your phone
5. See your app running!

## Frequently Asked Questions

### What is Expo Go?
Expo Go is a free app for your phone that lets you preview your mobile apps during development. Download it from the App Store (iPhone) or Google Play Store (Android).

### Do I need a Mac to build mobile apps?
No! With Turbocat and Expo, you can build mobile apps from any computer. You only need a Mac if you want to submit to the App Store.

### Can I use my existing web code?
Yes! If you have a web app in Turbocat, you can create a monorepo to share code between web and mobile. See the [Monorepo Guide](./05-monorepo-guide.md).

### How much does mobile development cost?
Mobile development uses Railway containers which cost approximately $5-10/month for typical usage. See [Cost Monitoring](./07-cost-monitoring.md) for details.
